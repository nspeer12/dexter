# RT_GestureRecognition
This demo was trained on The 20BN-jester Dataset V1 (https://20bn.com/datasets/jester/v1)
![](demo.gif)

# Getting the demo to work:

Clone the repository.
You might need to install some libs such as:
  - PyTorch
  - OpenCV2
  - Numpy
  - Pandas 
  - Torchvision
  
Run `demo.py`

Press `Q` to stop.
